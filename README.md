# The Gear Chronicles ⚙️

[![Deployed on Vercel](https://img.shields.io/badge/Deployed_on-Vercel-000000?style=for-the-badge&logo=vercel)](https://gear-chronicles.vercel.app/)

**[View Live Demo](https://gear-chronicles.vercel.app/)**

A visually immersive, interactive timeline exploring 2,000 years of gear and power transmission evolution. From the Roman tallow-lubricated wooden watermills to modern titanium lattice aerospace gears, this project visualizes the engineering breakthroughs that shaped the mechanical world.

## ✨ Features

- **Interactive Timeline**: A 7-era chronological journey featuring scroll-triggered reveal animations.
- **Dynamic SVG Animations**: Custom-built, CSS-animated SVG gear visuals demonstrating different mechanical principles (spur, bevel, worm drives, etc.).
- **Innovation Comparison**: A comprehensive data table comparing gear types, materials, contact types, and applications across history.
- **Frontier Research**: Insights into the future of power transmission, including additive manufacturing, AI in design, and ultra-precision EV gears.
- **Responsive Design**: A dark, industrial aesthetic optimized for both desktop and mobile viewing.

## 🛠️ Tech Stack

This project is built with a lightweight, zero-dependency stack:
- **HTML5**: Semantic structure and accessible markup.
- **CSS3**: Custom properties (variables), CSS Grid/Flexbox layouts, and keyframe animations.
- **Vanilla JavaScript**: Utilizing the `IntersectionObserver` API for performant scroll animations.

## 🚀 Getting Started

To run this project locally, you don't need any build tools or package managers.

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/gear-chronicles.git
   ```
2. Navigate to the project directory:
   ```bash
   cd gear-chronicles
   ```
3. Open `index.html` in your preferred web browser, or use a local development server (like VS Code's Live Server extension) for hot-reloading.
